User Flow:

#Profile

-User ingin melakukan Login:
1. Buka Aplikasi Mental  
2. Pilih bagian Profile 
3. Pilih login 
4. Maukkan Username dan Password 

-User ingin melakukan Sign up 
1. Buka Aplikasi Mental Health 
2. Pilih bagian Profile 
3. Pilih Sign Up
4. Isi form Profile 
5. Isi Username 
(6.) Jika user mengisi username yang sama dengan yang sudah pernah dibuat, maka akan kembali ke tahap 5, jika tidak, maka ke tahap selanjutnya 
7. Isi Password
8. Isi ulang Password

-User ingin keluar dari profil
1. Buka Aplikasi Mental Health  
2. Pilih bagian Profile 
3. Pilih Logout
(4.) Jika user tidak yakin untuk keluar, kembali ke tahap sebelumnya, Jika yakin, ke tahapan selanjutnya 
5. Tekan "Yes"

-User ingin melihat My Profile 
1. Buka Aplikasi Mental Health 
2. Pilih bagian Profile
3. Pilih bagian My Profile

-User ingin memperbaharui profile mereka
1. Buka Aplikasi Mental Health
2. Pilih bagian Profile
3. Pilih bagian My Profle 
4. Pilih update Profle
5. Pilih bagian yang ingin diperbaharui
6. Masukkan Perubahan baru
7. Tekan Save
(8.) Jika User tidak yakin untuk memperbaharui profile, kembali ke tahap 6, Jika yakin, lanjut ke tahap selanjutnya 
9. Tekan "Yes"
